 NULL-terminated list of pointers to arguments. It is used for signature validation before encode and send, and in case of a listener is defined, it will be used for signature validation before holding the received values.

 Incoming command listener. Will be called only if an incoming message is correctly received and if it pass the signature validation, so contained values can be trusted in error-free reception and type. Value itself is the listener duty ;)
