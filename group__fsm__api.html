◆ fsm_state

enum fsm_state

#include <fsm.h>

Valid states that a finite state machine can be in.

Enumerator
FSM_NEW 

single-pass state only, just for initialization and checks

FSM_PAUSE 

'Nothing to do' state. FSM possibly ended and no rearm bit was found

FSM_RUN 

FSM core, where the magic happens.

FSM_ERROR 

You're in a little trouble if you are here.

FSM_ALL 

Not a real state, just for stats as a cycle counter.
