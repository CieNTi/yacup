◆ _dbg

#define _dbg (   ...)

#include <debug.h>

Macro used to unify debugging messages.

This is a configurable macro:

  • If YACUP_DEBUG is defined, PRINTF macro will be used.
    • If PRINTF is not defined, standard printf will be used.
    • User can define PRINTF as a custom function. As example, instead of print to stdout, the string is sent through a serial port.
  • If YACUP_DEBUG is not defined, no debug function is used and no string storage allocation is performed.
Note
Read yacup/debug.h for complete information
