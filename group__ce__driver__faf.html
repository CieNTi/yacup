◆ ce_driver_faf()

int ce_driver_faf ( struct ce *  ce)

#include <fire-and-forget.h>

Driver initializer function for ce_driver_faf type CE.

This chat FSM will control input and output operations on a fire and forget type protocol, where the data is sent without any confirmation expectation or timeout, and when data is received there is no feedback to the remitent. It is the easiest form of protocol, where everything is supposed to be perfect, with no communication errors

cePointer to a ce to work with
